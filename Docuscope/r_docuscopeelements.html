DocuScope Interface

Refer to this page for a breakdown of the DocuScope application

Section Description Default View
Menu and Dictionary Menu functions and dictionary information.
Dictionary Navigation Panel Displays the loaded dictionary, and the hierarchical structure of the language categories in the dictionary.
Filter Options Filter clusters and dimensions by the number of instances of each.
Text View The right-side panel shows the text. When a category is selected on the left side panel, language patterns that correspond to the selected category are highlighted here.
Temporal View The lower section of the Text View window displays the temporal view, which allows you to see how language patterns are used over time (from the beginning to the end of the text).
