Dictionary Properties

Refer to this page for definitions of the dictionary properties.

Property Definition
Name The name of a category. No spaces are allowed, and they are automatically removed from the name. The preferred method is Upper Camel Case (e.g. FirstPerson).
Label This is a text label used in the help document, and is not necessary to include with every term.
Content Provide a brief description of the category with a few example patterns.
