Assign a Due Date

Add a date to a card to mark when to complete a task.

To add a due date

  1. Click the Card
  2. Select Due Date
  3. Enter Date and Time or Select a date using the Calendar
  4. Select a Reminder from the Set Reminder Menu
    Choose None to avoid being sent a reminder.
  5. Click Save
