Assign a Due Date

Add a due date to a card to indicate a deadline for the task.

  1. Click the Card.
  2. Select Due Date.
  3. Enter a Date and a Time or select a date using the Calendar.
  4. Select a Reminder from the Set Reminder menu.
    Select None to avoid being sent a reminder.
  5. Click Save.
